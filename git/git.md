### 常用命令

#### 1.拉取代码/资源

先在控制台进入到你想要克隆下来的路径
使用命令 cd  打到指定文件夹
使用命令 :`$ git clone url` （备注url即为远程仓库的地址）

#### 2.拉取代码/资源，即取回远程仓库的变化，并与本地分支合并

同理使用cd进入克隆下来的本地保存的地址

使用命令: `$ git pull`

#### 3.上传代码/资源，即上传本地指定分支到远程仓库

使用命令: `$ git push`
