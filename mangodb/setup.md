# MongoDB 安装

由于`MongoDB`不开源了，需要先设置下第三方仓库：

```bash
brew tap mongodb/brew
```

安装社区版：

```bash
brew install mongodb-community
```

启动`MongoDB`服务：

```bash
brew services start mongodb-community
```

停止`MongoDB`服务：

```bash
brew services stop mongodb-community
```

[MongoDB 官方网站](https://www.mongodb.com/zh)
